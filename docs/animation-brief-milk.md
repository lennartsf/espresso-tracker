## 0. Meta
- **Animation:** Milch aufschäumen → Mikroschaum (Flat White / Latte), Rancilio Silvia
- **Lernziel:** glänzenden Mikroschaum erzeugen („wet paint"), Tip richtig positionieren, kurz ziehen / lang rollen
- **Ansichten:** Side + Top kombiniert
- **Gesamtgefühl:** ruhig-kontrolliert
- **Ziel-Temperatur:** ~60–65 °C (Kanne zu heiß zum Halten = fertig)
- **Milchmenge / Kännchen:** _____ (korrigieren falls relevant)

## 1. Szene — was IMMER sichtbar ist (statisch)
- **Side-View:** Kännchen (leicht gekippt), feste Dampflanze von oben, Milchspiegel.
- **Top-View:** Milchoberfläche von oben, Lanzenposition (zwischen Mitte & Rand), Whirlpool.
- **Farb-/Stilwünsche:** Milch hell-gelb, Schaum weiß, Lanze grau, helle Karte.

## 2. Akteure — was sich BEWEGT
- **Kännchen:** bewegt sich (gesenkt/gehoben/gekippt) — **nicht** die Lanze.
- **Milchspiegel / Volumen:** wächst beim Ziehen (~20–30%).
- **Whirlpool:** dreht; Mitte sinkt leicht, Rand hebt (Trichter an der Lanze).
- **Schaumschicht:** entsteht beim Ziehen, wird beim Rollen fein.

## 3. Phasen (Entwurf — Anteile/Details korrigieren)

```
PHASE 1 — Purge / Ansetzen   (~5 %)
- Tip: knapp unter Oberfläche, leicht außermittig (zwischen Mitte & Rand)
- Kännchen: leicht gekippt
- Effekt: kurz abdampfen, ansetzen
- Caption: „Lanze kurz abdampfen, Tip knapp unter die Oberfläche setzen."
- korrigieren:

PHASE 2 — Ziehen / Stretch   (~25 %)
- Tip: KNAPP unter Oberfläche → Luft rein („tsch-tsch")
- Volumen: wächst ~20–30 %; Kännchen langsam senken, Tip bleibt an Oberfläche
- Whirlpool: beginnt
- Caption: „Tip an der Oberfläche — Luft einarbeiten, bis das Volumen wächst."
- häufiger Fehler: zu lange ziehen → zu viel/grober Schaum
- korrigieren:

PHASE 3 — Rollen / Textur   (~55 %)
- Tip: etwas tiefer eintauchen → Whirlpool, KEINE neue Luft
- Effekt: Blasen werden zu Mikroschaum, bis ~60–65 °C
- Caption: „Tip tiefer — rollen lassen, bis die Blasen verschwinden (Mikroschaum)."
- häufiger Fehler: Tip zu tief von Anfang an → gar kein Schaum
- korrigieren:

PHASE 4 — Abschluss   (~15 %)
- Dampf aus, Lanze raus
- Kännchen schwenken / auf Tisch klopfen → polieren
- Caption: „Dampf aus, schwenken und klopfen — glänzend wie nasse Farbe."
- korrigieren:
```
